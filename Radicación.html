La radicación es la operación que permite hallar un valor que multiplicado tantas veces como lo indica el índice, dé el valor que se encuentra dentro del radical, el cual recibe el nombre de radicando. Para lo anterior se define:

\(\displaystyle \sqrt[n]{a^m}=a^{\frac{m}{n}} \)

donde:
\(a\) es la base
\(m\) es el exponente
\(n\) es el índice.


TeoremasLos teoremas de los exponentes también se aplican a radicales, ya que se expresan como exponentes fraccionarios.

\(\displaystyle \sqrt[n]{a\cdot b\cdot c}=\sqrt[n]{a}\cdot \sqrt[n]{b}\cdot  \sqrt[n]{c}\)

\(\displaystyle \sqrt[n]{\frac{a}{b}}=\frac{\sqrt[n]{a}}{\sqrt[n]{b}}\)

\(\displaystyle \sqrt[n]{\sqrt[m]{a}}=\sqrt[(n\cdot m)]{a}\)


Ejemplo. Calcular \(\displaystyle \sqrt[5]{3}\cdot \sqrt[5]{2}\cdot \sqrt[5]{5}\)

\(\displaystyle \sqrt[5]{3}\cdot \sqrt[5]{2}\cdot \sqrt[5]{5}=\sqrt[5]{3\cdot 2\cdot 5}=\sqrt[5]{30}\)


Ejemplo. Calcular \(\displaystyle \frac{\sqrt[3]{10}}{\sqrt[3]{2}}\)

\(\displaystyle \frac{\sqrt[3]{10}}{\sqrt[3]{2}}=\sqrt[3]{\frac{10}{2}}=\sqrt[3]{5}\)
