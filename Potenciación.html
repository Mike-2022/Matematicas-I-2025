Potenciación
Es la operación en la cual la cantidad llamada base se debe multiplicar por ella misma las veces que lo indique el exponente. De lo anterior se define:

\(\displaystyle a^n =a\cdot a\cdot a ...\,\, n\, veces\)

donde:
\(a\) es la base
\(n\) es el exponente.

\(\displaystyle a^{-n} =\frac{1}{a^n}=\frac{1}{a\cdot a\cdot a ...}\)

Ejemplo 1. Desarrolla 52

Solución
Al ser el exponente 2, la base 5 se debe multiplicar 2 veces ella misma:

\(5^2= (5)(5)= 25\)

Por tanto, el resultado de 52= 25

Ejemplo 2. ¿Cuál es el resultado de \(\displaystyle \left(\frac{1}{2}\right)^3\)?

Solución
La fracción se debe multiplicar 3 veces por ella misma.

\(\displaystyle \left(\frac{1}{2}\right)^3=\left(\frac{1}{2}\right) \left(\frac{1}{2}\right) \left(\frac{1}{2}\right)=\left(\frac{1}{8}\right)\)

Ejemplo 3. Desarrolla 3−4.

Solución
Se aplica la definición y luego se desarrolla 3 para obtener el resultado.

\(\displaystyle 3^{-4} =\frac{1}{3^4}=\frac{1}{3\cdot 3\cdot 3 \cdot 3}=\frac {1}{81}\)

Cuando un número negativo se eleva a una potencia par, el resultado es positivo, pero si se eleva a una potencia impar, el resultado es negativo.

Ejemplo 4. ¿Cuál es el resultado de (−6)4?

Solución
La potencia es par, por tanto, el resultado es positivo.

\((-6)^4=(-6)(-6)(-6)(-6) =1296\)

Ejemplo 5. Desarrolla  (−3)5.

Solución
La potencia es impar el resultado será negativo.

\((-3)^5=(-3)(-3)(-3)(-3)(-3) =-243\)


Teoremas de la potenciación
Producto de potencias\(\displaystyle a^m\cdot a^n =a^{(m+n)}\)

Ejemplo. Calcular \(2^3\cdot 2^2\)

Solución
\(2^3\cdot 2^2⋅=2^{(3+2)}=2^5=32\)


Cociente de potencias\(\displaystyle \frac{a^m}{a^n} =a^{(m-n)}\)

Ejemplo. Calcular \(\displaystyle \frac{3^5}{3^2}\)

Solución
\(\displaystyle \frac{3^5}{3^2}=3^{(5-2)}=3^3=27\)


Potencia cero\(\displaystyle a^0 =1\)


Potencia de potencia\(\displaystyle (a^m)^n =a^{(m\cdot n)}\)

Ejemplo. Calcula \((a^3)^2\)

Solución
\((a^3)^2=a^{(3\cdot 2)}=a^6=4 096\)


Potencia de un producto\(\displaystyle (a\cdot b\cdot c)^m =a^m\cdot b^m\cdot c^m\)

Ejemplo. Calcula \((2\cdot 3\cdot 5)^2\)

Solución
Se realiza el producto de \(2\cdot 3\cdot 5= 30\) y después se eleva \(30^2 = 900\)

O de otra manera 
\(2^2\cdot 3^2\cdot 5^2= 4\cdot 9\cdot 25=900\)


Potencia de un cociente\(\displaystyle \left(\frac{a}{b}\right)^m=\frac{a^m}{b^m}\)

Ejemplo. Calcula \(\displaystyle \left(\frac{3}{4}\right)^2\)

Solución
\(\displaystyle \left(\frac{3}{4}\right)^2=\frac{3^2}{4^2}=\frac{9}{16}\)
